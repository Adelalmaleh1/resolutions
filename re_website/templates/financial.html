Valuation Management

Indicative Value Check

  • Get instant market value estimate - backed by up to data transaction data
  • Automatically request for indicative values from all panel valuers

Process Controls

  • Single point to order, track, manage and retrieve valuation reports
  • Standardised digitally signed, and un-editable valuation reports for all panel valuers

Valuers Performance Management

  • Intuitive web based interface for effective data entry and file attachments
  • Seamless synchronisation with VMS Business Edition (Subscription Service for Valuers)

Risk Management

  • Automated valuation report compliance checks
  • Automated quality assurance checks

Panel Management

  • Effectively Manage and change panel valuers
  • Instantly manage and implement valuation policies to all panel value

Porfolio Revaluation

Revalue entire portfolios instantly by leveraging the very latest transaction data and in depth analytics

  • The system is designed to extract information from an Excel file through the correspondence of specific column headings. This enables the system to provide a “Market Value Estimate” based on the Bank’s specific risk parameters.
  • From the results derived or based on existing parameters, a request for indication from panel valuers can be both automatically or manually triggered for verification or test purposes.
